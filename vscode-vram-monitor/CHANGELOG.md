# Changelog

All notable changes to the VRAM Monitor & Ollama Manager extension will be documented in this file.

## [0.1.0] - 2025-12-14

### Added
- Initial release
- Real-time VRAM monitoring via `vram_manager.py`
- Status bar indicator with color-coded thresholds
- Interactive dashboard sidebar with GPU stats, models, and processes
- Ollama model management (load/unload)
- Configurable visibility settings
- Diagnostic commands for troubleshooting
- Graceful degradation when Ollama offline or no GPU detected
